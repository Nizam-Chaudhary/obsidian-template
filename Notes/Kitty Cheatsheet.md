The default keybindings for the Kitty terminal emulator.

**Note:** Kitty relies heavily on the `Ctrl` + `Shift` modifier for its commands to avoid conflicting with standard shell shortcuts (like `Ctrl+C` for interrupt).

### ü™ü Windows (Splits/Panes)

In Kitty terminology, "panes" are called "windows."

| **Action**               | **Shortcut**                                |
| ------------------------ | ------------------------------------------- |
| **New Window**           | `Ctrl` + `Shift` + `Enter`                  |
| **Close Window**         | `Ctrl` + `Shift` + `w`                      |
| **Next Window**          | `Ctrl` + `Shift` + `]`                      |
| **Previous Window**      | `Ctrl` + `Shift` + `[`                      |
| **Move Window Forward**  | `Ctrl` + `Shift` + `f`                      |
| **Move Window Backward** | `Ctrl` + `Shift` + `b`                      |
| **Resize Windows**       | `Ctrl` + `Shift` + `r` (enters resize mode) |

### üìë Tabs

| **Action**            | **Shortcut**                     |
| --------------------- | -------------------------------- |
| **New Tab**           | `Ctrl` + `Shift` + `t`           |
| **Close Tab**         | `Ctrl` + `Shift` + `q`           |
| **Next Tab**          | `Ctrl` + `Shift` + `Right Arrow` |
| **Previous Tab**      | `Ctrl` + `Shift` + `Left Arrow`  |
| **Move Tab Forward**  | `Ctrl` + `Shift` + `.` (period)  |
| **Move Tab Backward** | `Ctrl` + `Shift` + `,` (comma)   |
| **Set Tab Title**     | `Ctrl` + `Shift` + `Alt` + `t`   |

### üß≠ Navigation & Scrolling

| **Action**            | **Shortcut**                                        |
| --------------------- | --------------------------------------------------- |
| **Scroll Line Up**    | `Ctrl` + `Shift` + `Up Arrow`                       |
| **Scroll Line Down**  | `Ctrl` + `Shift` + `Down Arrow`                     |
| **Scroll Page Up**    | `Ctrl` + `Shift` + `Page Up`                        |
| **Scroll Page Down**  | `Ctrl` + `Shift` + `Page Down`                      |
| **Scroll to Top**     | `Ctrl` + `Shift` + `Home`                           |
| **Scroll to Bottom**  | `Ctrl` + `Shift` + `End`                            |
| **Browse Scrollback** | `Ctrl` + `Shift` + `h` (opens buffer in pager/less) |

### üìê Layouts

Kitty has different layouts (Tall, Fat, Grid, etc.) for arranging windows.

- **Next Layout:** `Ctrl` + `Shift` + `l`
- **Restore Last Layout:** `Ctrl` + `Shift` + `Space`

### üìã Clipboard & Font Size

- **Copy to Clipboard:** `Ctrl` + `Shift` + `c`
- **Paste from Clipboard:** `Ctrl` + `Shift` + `v`
- **Increase Font Size:** `Ctrl` + `Shift` + `+` (or `=`)
- **Decrease Font Size:** `Ctrl` + `Shift` + `-`
- **Reset Font Size:** `Ctrl` + `Shift` + `Backspace`

- **Reload Config:** `Ctrl` + `Shift` + `F5`
